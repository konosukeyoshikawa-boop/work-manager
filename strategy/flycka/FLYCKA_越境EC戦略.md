## 0. 最重要リスク: 中国仕入れ品の越境販売問題

### 問題の構造

FLYCKAの商品はOEM・セレクト型（中国工場の既製品から選定、FLYCKAタグを付けて販売）。
台湾・香港の消費者は淘宝/Shopeeで同等品を1/3〜1/8の価格で購入できる可能性がある。

| チャネル | 類似品の台湾価格 | FLYCKAとの比 |
|---|---|---|
| 淘宝直接 | TWD 150〜400 | 1/3〜1/6 |
| SHEIN/Shopee中国セラー | TWD 100〜350 | 1/4〜1/8 |
| 台湾ローカルブランド | TWD 300〜800 | 1/1〜1/2 |
| **FLYCKA想定** | **TWD 1,700〜2,700** | **最高価格帯** |

### 中国ECアクセス容易性（8市場調査結果）

| 市場 | スコア(5段階) | 詳細 |
|---|---|---|
| 香港 | 5（非常に容易） | 淘宝が生活インフラ化。Cainiao集荷1,000拠点 |
| 台湾 | 5（非常に容易） | 中国語ネイティブ、代購文化定着 |
| シンガポール | 5（非常に容易） | 華人75%、越境注文60%が中国発 |
| 韓国 | 4 | AliExpress MAU 912万。ただし淘宝直接利用は少 |
| マレーシア | 4 | 華人23%が中国EC積極利用 |
| タイ | 4 | Temu/SHEIN浸透中。規制圧力あり |
| フィリピン | 3 | 淘宝直送なし、言語障壁大 |
| ベトナム | 2（困難） | Temu/SHEIN政府規制で遮断 |

### ジレンマ

- 購買力が高い市場（台湾・香港・SG） = 中国ECアクセスも容易（リスク大）
- 中国ECの壁が高い市場（ベトナム・フィリピン） = 購買力が低い（FLYCKA価格帯に合わない）
- 両方を満たす市場がほぼ存在しない

### 経営判断（2026-03-13）

**リスクを認識した上で、台湾・香港で少額テストを実行する。**

理由:
1. 分析だけでは判断できない。実データで検証する方が合理的
2. テスト費用は月5〜10万円。失敗しても損失は限定的
3. デザインのテイスト・世界観で差別化できる可能性は否定できない
4. 台湾・香港のMeta ASCでROAS 3.0以上が出れば、理論上のリスクは実際には顕在化していないことの証明になる

### テストで検証すべき仮説

| # | 仮説 | 検証方法 | 判断基準 |
|---|---|---|---|
| H1 | 台湾・香港でROAS 3.0以上が出る | Meta ASC越境広告 | 4週間でROAS 3.0未満なら仮説棄却 |
| H2 | 消費者は「淘宝で同じもの」を探さない | 購入後アンケート + レビュー監視 | 「淘宝」「taobao」メンション出現で警戒 |
| H3 | 「日本ブランド」のプレミアムで購入する | ASCの国別CVR比較 | 台湾CVR > 国内CVRの50%なら合格 |
| H4 | 返品率が国内と同等以下 | 返品データ | 返品率15%以上で仮説棄却 |

## 2. なぜ越境ECか — 3つの戦略的意義

### 2.1 短期: 利益構造の改善
- 円安（1USD=150円台）で海外顧客から見たFLYCKAの価格は**割安**
- 台湾のMeta広告CPMは$0.80〜$3（日本$6.73の**1/3〜1/8**）
- 日本ブランドの台湾・香港向けMeta ASCでROAS 300〜914%の実績（複数社）
- 国内の損益分岐ROAS 2.5〜3.0が、円安＋低CPMで**実質1.5〜2.0に低下**

### 2.2 中期: 事業バリュエーション向上
- 「越境EC実績あり」は**M&A評価で加点要素**（売却目標3億円に直結）
- 国内1チャネル依存からの脱却 → リスク分散
- TAMが国内1.8兆円 → アジア太平洋2.4兆円に拡大

### 2.3 批判的検討で明らかになった留意点（v1から継承）
- CPMが安い ≠ CPAが安い。越境ECは「初見ブランド」「試着不可」「配送遅延」のCVRハンデがある
- 国内ROASの安定化とのリソーストレードオフを常に意識する
- ただし**月15〜30万円の投資規模であれば、国内施策への影響は軽微**

- **既存のMeta広告運用ノウハウ・クリエイティブがほぼそのまま転用可能**

**価格ポジション**: 4,000〜6,000円 ≒ TWD 850〜1,260。台湾市場では中価格帯（Wacoalより安く、ローカルブランドよりやや高い）で、最もボリュームが大きいゾーン。

**リスク**
- 人口2,300万人。ターゲット（20-34歳女性）は約300万人。天井が低い可能性

### 3.3 v1からの変更: なぜタイを格下げしたか

v1ではタイを第2候補（S評価）としていたが、以下の新情報により格下げ:

1. **関税**: 繊維製品の関税が最大60%（v1時点では未調査だった）
2. **実効価格**: 4,000円の商品 → 関税後6,400円。中価格帯から高価格帯に跳ね上がる
3. **代替**: 香港（関税ゼロ）とシンガポール（関税ゼロ）が同等以上の条件を持つ

タイ市場自体のポテンシャル（人口7,200万人、親日、Meta広告有効）は依然として高いが、関税障壁を考慮すると**Phase 3以降で現地パートナー or 現地倉庫を活用する際に再検討**が適切。

### 4.2 各プラットフォーム詳細

#### Shopee（最重要チャネル）

**なぜShopeeか:**
- 下着カテゴリの需要が前年比**+1,994%増**（Shopee社公式データ）
- SHIROHATO（白鳩）が下着カテゴリで3ヶ月でオーダー数**20倍**の実績
- 初期費用ゼロ、**90日間コミッション無料特典**あり
- SLS（Shopee Logistics Service）で国際配送を一元管理

**v1ではShopee先行を否定した理由と、v2での再評価:**

| v1の論点 | v2の反論 |
|---|---|
| 下着はブランド世界観で買う → Shopeeでは伝わらない | Shopeeの「Mall」出店なら公式ストア扱い。SHIROHATOも世界観を維持して成功 |
| Shopeeで売れなかった場合の検証が曖昧 | Meta ASC越境広告を並行テストすることで、Shopee/Shopify両面から検証可能 |
| PEACH JOHNはShopeeではなくWorldShoppingBIZを選んだ | PEACH JOHNの選択は2022年頃。2024-2025年のShopee下着カテゴリの急成長を反映していない |

**結論**: v1の「Shopify先行」は正しいブランド戦略だが、月15〜30万円の予算制約下では**Shopeeの集客力を活用しつつ、Shopify Marketsを並行運用**するのが合理的。

**Shopee出店の具体要件:**
- Japan Seller Program（日本法人/個人で出店可能）
- 対応市場: 台湾、シンガポール、マレーシア、タイ、フィリピン、ベトナム
- 必要書類: 本人確認書類、銀行口座
- SLS対応: 日本国内集荷 → Shopee物流で国際配送
- 広告: Product Search Ads（CPC）、Discovery Ads、フラッシュセール、バウチャー

#### Shopify Markets（ブランド体験チャネル）

**対応範囲:**
- 多通貨: 130通貨対応
- 多言語: 50言語（Translate & Adaptアプリ）
- 関税計算: 全プラン対応、手数料0.5%
- 決済: Shopify Payments（香港・SG対応）、KOMOJU（台湾）
- SEO: hreflang自動生成、サブフォルダ構造推奨

**制約:**
- Managed Markets（旧Pro）は**日本拠点では利用不可**（米国フルフィルメント必須）
- 集客は完全に自力（Meta広告 or KOL必須）
- 台湾向けはShopify Paymentsが未対応のためKOMOJU等が必要

#### WorldShopping BIZ（低コスト補完チャネル）

**概要:** 既存Shopifyサイトにタグ1行を追加するだけで、海外顧客が現地通貨・現地決済で購入可能に。

**メリット:**
- 事業者手数料ゼロ（購入者がサービス手数料を負担）
- 導入工数が極小（JSタグ1行）
- 関税・国際配送をWorldShoppingが代行
- Meta広告との連携でCPA 90%改善の実績あり

**デメリット:**
- 能動的な集客ができない（受動的）
- 購入者が手数料を負担するため実質的な購入価格が上がる
- CV計測がやや複雑

#### TikTok Shop（将来チャネル）

- 2025年6月にJapan越境対応ローンチ
- 手数料7%（新規出店3ヶ月は3%）
- ショートビデオ × ECの親和性は高い（下着のフィッティング動画等）
- ただし越境対応はまだ未成熟。Phase 3以降で検討

| **合計** | | **22〜33万円** |

**メリット**: ブランド体験を完全コントロール。v1の戦略思想を継承。
**リスク**: 集客が広告・KOLに完全依存。Shopeeの集客力を使えない。

### 4.4 推奨: パターンA（攻めの2面展開）

**理由:**
1. Shopeeの90日コミッション無料を活用し、リスクなくマーケットプレイスをテスト
2. Shopify Marketsで香港（関税ゼロ）をカバーし、ブランド体験を維持
3. Meta ASC越境広告で、国内とは独立したROASデータを取得
4. 3チャネルの同時テストにより、**「どのチャネル × どの市場が最もROASが高いか」を最速で判断可能**

## 5. マーケティング戦略

### 5.1 Meta広告の越境展開（最重要施策）

**v2での最大の追加要素。** 日本ブランドの台湾・香港向けMeta ASCで複数の成功事例が判明。

#### 成功事例

| 企業 | 商材 | 配信国 | 成果 |
|---|---|---|---|
| デイトナ（FREAK'S STORE） | アパレル | 台湾・香港 | **売上9.2倍、ROAS 800%+** |
| フェイバリット | レディースアパレル | 台湾・香港 | **ROAS国内比1.5倍、CPA国内比70%** |
| 愛鷹製作所 | 金属製品 | 台湾・香港・米国 | **ROAS最大914%** |
| ディールデザイン | ジュエリー | 香港 | **売上93%増、ROAS 300%** |

#### 5社共通の成功パターン
1. **Advantage+ カタログ広告（ASC）を使用**
2. **台湾・香港から開始**
3. **月5万円の少額テストからスケール**
4. **国内カタログをそのまま流用**（日本語でも成果が出る）
5. **ターゲティングはAIに委任**（ブロードターゲティング）

#### FLYCKAのMeta ASC越境設定

| 設定項目 | 推奨値 |
|---|---|
| キャンペーンタイプ | Advantage+ Shopping Campaign |
| 配信国 | 台湾 + 香港 |
| 予算 | **月10万円**（日予算3,300円）からスタート |
| カタログ | 既存の日本語カタログをそのまま使用 |
| クリエイティブ | 国内の勝ちクリエイティブを流用（物撮り中心） |
| ターゲティング | ブロード（AIに最適化を委任） |
| 除外 | 日本を除外（国内キャンペーンと分離） |
| Pixel | 通貨コードをJPY固定 → 動的に変更 |
| CAPI | Shopify × Meta連携で設定（ROAS 30%改善報告あり） |

**なぜ日本語カタログでOKか:**
- 台湾・香港では「日本語=本物の日本ブランド」として信頼される
- 現地語翻訳より日本語のほうがCVRが高い事例あり
- テスト段階では翻訳コストをかけず、**ROASの初期データ取得を最優先**

### 5.2 下着広告のリジェクト対策（v1から継承・補強）

**Metaの自動審査で「成人向けコンテンツ」として誤検知されるリスクは海外配信でも同等。**

| 対策 | 詳細 |
|---|---|
| クリエイティブ | 単品物撮り（白背景）を基本。着用カットは上半身全体 |
| テキスト | 「sexy」等の表現を一切使わない。機能訴求に徹する |
| 台湾でのバックアップ | LINE Ads（MAU 2,200万人） or Shopee Ads（リジェクトなし） |
| アカウント管理 | ビジネス認証完了。異議申し立てフローを事前構築 |

### 5.3 広告チャネル優先順位（v2）

```
優先度1: Meta ASC越境広告（台湾・香港向け。月10〜20万円。ROASデータ取得最優先）
優先度2: Shopee Ads（台湾Shopee内CPC広告。月3〜5万円）
優先度3: Instagram KOL（台湾マイクロKOL 1〜2名。月5〜10万円）
優先度4: LINE Ads（Meta不安定時の代替。台湾LINE MAU 2,200万人）
```

**v1との違い**: v1はKOL起点だったが、v2ではMeta ASCを最優先に変更。理由は:
- KOLは効果のばらつきが大きく、初期テストには不向き
- Meta ASCは予算コントロールが容易で、ROAS計測が正確
- 成功事例のほぼ全てがASC起点

### 5.4 KOL戦略（v1から継承、ただし優先度を下げた）

| 項目 | 方針 |
|---|---|
| 起用人数 | マイクロKOL（1〜10万フォロワー）を**1〜2名/月** |
| 選定基準 | エンゲージメント率3%以上、台湾20-30代女性フォロワー |
| 報酬 | サンプル提供 + 投稿報酬（7.5〜15万円/投稿） |
| トラッキング | KOL限定クーポンコード |
| タイミング | Meta ASCで初期ROASを確認後（Phase 2以降） |
