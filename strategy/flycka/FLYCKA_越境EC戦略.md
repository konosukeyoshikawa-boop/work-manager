## 2. なぜ越境ECか

### 2.1 目的
- **販路拡大**: 国内市場に加え、アジア圏の女性顧客層を開拓
- **CPM引き下げ**: 台湾のMeta広告CPMは$0.80〜$3（日本$6.73の1/3〜1/8）

### 2.2 批判的検討で明らかになった留意点
- CPMが安い ≠ CPAが安い。越境ECは「初見ブランド」「試着不可」「配送遅延」のCVRハンデがある
- 国内ROASの安定化とのリソーストレードオフを常に意識する必要がある
- 同じリソースを国内改善（クリエイティブ改善・LPO・CRM強化）に投じた場合のリターンと比較すべき

### 3.2 台湾を第1候補とする理由

**市場環境**
- Facebook利用者1,710万人（人口の93.6%）、Instagram利用者1,220万人（52.6%）
- IG女性比率56%、25-34歳が最大セグメント（410万人）
- LINE MAU 2,200万人（人口の約94%）
- ソーシャル広告市場約9.6億ドル（2025年）

**下着ブランドとしての適合性**
- 体型が日本人とほぼ同一 → サイズ表をほぼ流用可能（在庫リスク最小）
- PEACH JOHNがWorldShoppingBIZ経由で海外展開し、アジア圏が最多注文地域
- aimerFeelが台湾に6店舗展開済み → 市場存在の実証
- 薬事規制なし。繊維ラベル表示のみで参入可能

**広告チャネルの冗長性（台湾最大の利点）**
- Meta広告がリジェクトされてもLINE Ads（MAU 2,200万人）にシフト可能
- Shopee Adsも利用可能（リジェクトリスクなし）
- 4チャネルすべてが強い稀有な市場

### 3.3 台湾市場のリスク
- 人口2,300万人。ターゲット（20-34歳女性）は約300万人。天井が低い可能性
- 台湾で成功しても、タイ（7,200万人）やインドネシア（2.7億人）へのスケーラビリティは別問題

## 4. チャネル戦略：なぜShopify先行か

### 4.1 Shopee vs Shopifyの検討経緯

当初はShopee先行（低コスト・低リスク）を推奨したが、以下の議論を経てShopify先行に決定。

**Shopee先行を否定した理由**
- 下着は信頼とブランド世界観で買うカテゴリ。Shopeeのモール環境では「安い順ソート」「類似商品の羅列」の中に埋もれる
- Shopeeで検証できるのは「価格競争力があるか」であって「ブランドとして受け入れられるか」ではない
- Shopeeで売れなかった場合、「台湾で需要がない」のか「Shopeeというチャネルが合わない」のか判別できない
- PEACH JOHNはWorldShoppingBIZ（自社EC越境）を選び、Shopeeではなかった

**下着の購買意思決定構造**

```
化粧品:  成分・効果 → レビュー → 価格比較 → 購入
下着:    世界観・デザイン → 信頼構築 → サイズ確認 → 購入
         ^^^^^^^^^^^^^^^^   ^^^^^^^^
         Shopeeでは伝わらない  Shopeeでは構築できない
```

**結論**: FLYCKAが世界観で勝負するブランドである以上、Shopify先行が正しい検証方法。

### 4.2 Shopify先行の集客設計

Shopify最大の課題は「自分で集客する必要がある」こと。これをInstagram × KOLで解決する。

```
正しい導線:
  ① Instagramで世界観を発信（オーガニック + KOL）
  ② 興味を持った人がプロフィールリンクからShopifyへ
  ③ Shopifyで世界観を維持したまま購買体験を完結
```

この流れのメリット:
- Meta広告のリジェクト問題を回避（オーガニック投稿は広告審査対象外）
- ブランド世界観を起点にした集客ができる
- Shopeeの価格競争に巻き込まれない
- KOLの着用投稿 → フォロワーの購買という下着に最も自然な導線

### 4.3 コスト構造の比較

| 項目 | Shopee台湾 | Shopify自社EC |
|---|---|---|
| 月額固定費 | 0円 | 約$39〜$105/月 |
| 販売手数料 | コミッション14% + 決済2.5% ≒ 16.5% | 決済5.8〜5.9%（通貨換算込み） |
| 関税徴収 | Shopee側で処理 | Avalara経由 +0.5% |
| 販売価格3,000円の商品での手取り | 約2,505円 | 約2,811円（+306円/個） |

→ Shopifyの方が1個あたり約10%マージンが大きい。

## 5. マーケティング戦略

### 5.1 広告チャネル優先順位

```
優先度1: Instagram オーガニック + KOL（ブランド認知構築。広告審査を経由しない）
優先度2: Meta広告（物撮り中心で慎重にテスト。台湾IG 1,220万人にリーチ）
優先度3: LINE Ads（Metaが不安定な場合の代替。台湾LINE MAU 2,200万人）
優先度4: Google P-MAX（検索需要の刈り取り）
```

### 5.2 Meta広告の下着ブランド特有リスク

**問題**: Metaの広告ポリシーでは下着広告は「許可」されているが、AIによる自動審査で頻繁に「成人向けコンテンツ」として誤検知・リジェクトされる。

**事例**:
- Pantee: コンテンツが継続的にブロックされ成長が停滞
- Curvy Kate: 広告が不公平に禁止される事態
- 複数のブランドが一貫性のないポリシー適用に苦しんでいる

**対策**:
| 対策 | 詳細 |
|---|---|
| クリエイティブ | 単品物撮り（白背景）を基本。着用カットは上半身全体。身体の特定部位にフォーカスしない |
| テキスト | 「セクシー」等の表現を一切使わない。機能訴求（フィット感・素材・快適さ）に徹する |
| UGC活用 | 顧客レビュー動画をクリエイティブに。第三者視点でポリシー違反リスクが下がる |
| アカウント管理 | 1広告アカウントに集中しない。ビジネス認証を完了。異議申し立てフローを事前構築 |

**成功事例**: Krvvy社はMeta広告で75日間で7桁（100万ドル超）の売上を達成。物撮り中心+10本以上の高品質動画+ブロードターゲティングで成功。

### 5.3 KOL戦略（最重要チャネル）

| 項目 | 方針 |
|---|---|
| 起用人数 | マイクロKOL（1〜10万フォロワー）を3〜5名 |
| 選定基準 | フォロワー数よりエンゲージメント率（3%以上）とフォロワー属性（台湾20-30代女性） |
| 選定ツール | Kolr（AI分析）or 台湾特化代理店 |
| 報酬 | サンプル提供 + 投稿報酬。1名あたり7.5〜38万円/投稿 |
| 投稿内容 | 着用レビュー・サイズ感レポート・開封動画。Shopifyサイトへのリンク必須 |
| トラッキング | KOL限定クーポンコードで売上計測 |

### 5.4 Shopee Adsについて（参考情報）

Shopifyと並行してShopeeを運用する場合に利用可能。すべてCPC（クリック課金）。
- Product Search Ads: 検索結果ページ上位表示
- Discovery Ads: トップページ「Daily Discover」等に表示
- 無料施策: バウチャー、フラッシュセール、Shopee Live、Shopee Video

Shopee内広告は下着カテゴリを正式に扱っており、Meta広告のようなリジェクト問題がほぼない。

## 6. 化粧品との規制面の違い

| 観点 | 化粧品 | 女性下着 |
|---|---|---|
| 規制 | 薬事規制が極めて重い（成分届出・許可制） | 繊維製品のラベル表示のみ。参入障壁が低い |
| 広告 | 比較的自由 | Meta広告でリジェクトリスクあり |
| サイズ | 不要 | 体型差への対応が必須。サイズ展開=在庫リスク |
| 返品 | 未開封なら再販可 | 衛生上、試着後の返品=廃棄。損失直結 |
| 文化的配慮 | 低い | 宗教・文化によるデザイン制約（ムスリム圏） |
| 競合 | 韓国コスメが強い | Wacoal（日本）・Triumph（欧州）がアジア全域で確立 |

### 6.1 台湾の繊維製品ラベル表示要件（JETRO）

台湾の「商品標示法」に基づき、以下5項目の中国語（繁体字）表示が必須:
1. 輸入者情報（氏名、住所、電話番号）と外国製造者名
2. 原産地
3. サイズ
4. 繊維成分
5. 洗濯方法およびアイロンの方法

有害物質基準: 遊離ホルムアルデヒド、特定芳香族アミン染料、付属品の鉛・有機錫等

関税: 繊維製品10.5〜12% + 営業税（VAT）5% + 貿易サービス税（CIFの0.04%）

## 11. 議論の経緯

本戦略書は以下の段階的な議論を経て策定された:

1. **初期提案（化粧品前提）**: アジア圏越境ECの国選定とTODOを整理
2. **商材修正**: FLYCKAが女性向け下着ブランドであることを反映し、規制・広告リスク・サイズ問題・文化的配慮の観点で全面再検討
3. **チャネル検討**: Shopee × Shopify並行運用のメリット・デメリットを分析し、「並行ではなく順序（Shopee先行）」を仮決定
4. **Shopee Ads詳細**: Shopee内のマーケティング施策（CPC広告・バウチャー・Live・Video）を調査
5. **Meta広告の台湾有効性**: 台湾のSNS普及率・CPM・下着ブランドのリジェクト問題を詳細分析
6. **戦略への批判**: 7つの観点で自己批判を実施（目的の解像度・市場規模・チャネル適合性・リソース分散・サイズ検証・撤退基準・タイミング）
7. **チャネル再決定**: 「下着は世界観で買うカテゴリ」という論点からShopee先行を撤回し、Shopify先行に決定
8. **最終TODO策定**: Shopify先行での具体的な実行計画を策定
